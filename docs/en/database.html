Database

Description of the database

The database consists of one unique table (denormalized), with one row representing one company.

The current version contains 63652 rows with the information described in the table below.

The completion rate indicates the number of companies for a given informationif available.

Column Completion rate (%) Description
wiki_name The name of the company as per the Wikipedia article
name 97 The official name of the company
isin 0 The International Securities Identification Number (ISIN), that is a code that uniquely identifies a specific securities issue.
revenue_amount 14 The company's revenues.
revenue_currency 12 The currency in which the revenues amount is expressed.
revenue_year 14 The last year at which the revenues were known.
revenue_src_url 5 The URL of the page from which the revenue was extracted.
num_employees 33 The company's number of employees.
num_employees_year 13 The last year at which the number of employees was known.
net_income_amount 8 The company's net income.
net_income_currency 6 The currency in which the net_income_amount is expressed.
net_income_year 8 The last year at which the net income was known.
net_income_src_url 0 The URL of the page from which the revenue was extracted, if different from revenue_src_url.
operating_income_amount 5 The company's operating income.
operating_income_currency 4 The currency in which the operating_income_amount is expressed.
operating_income_year 4 The last year at which the operating income was known.
operating_income_src_url 0 The URL of the page from which the revenue was extracted, if different from revenue_src_url.
industry 85 The type of industry the company is operating in.
hq_country 24 The country in which the company's headquarters are located.
